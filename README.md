# study-docker-typescript

Docker上でTypeScriptの基礎を学習するためのリポジトリです。

## 学習目的
- Dockerを使ってNode.js / TypeScript環境を動かす
- TypeScriptの基本的な型の考え方を理解する
- Git / GitHub を使った学習管理に慣れる

---

## 環境
- OS：Windows
- Docker Desktop
- Node.js（Dockerコンテナ内）
- TypeScript
- tsx（watch実行）

---

## 起動方法
```bash
docker compose up --build
```
